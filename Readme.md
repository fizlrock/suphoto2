Сервис для хранения данных о мероприятиях и организаторах. 

Примененные технологии:
  - Gradle
  - Java 21 
  - Spring Data JPA
  - Spring Web MVC
  - Liquibase
  - JUnit 5
  - Mapstruct
  - OpenAPI generator

[Старая версия проект (Maven, Java 11)](https://github.com/fizlrock/suphoto)
